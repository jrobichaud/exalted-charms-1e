# Prime - Documentation Understanding

Get a comprehensive understanding of the Exalted charm documentation and optionally dive deep into a specific charm tree.

## Instructions

### 1. Explore the Documentation

Investigate and understand the documentation structure:

- Project organization (Exalt types, castes, abilities)
- How RST files reference Mermaid diagrams
- Mermaid formatting conventions for charms
- Where different Exalt types are documented
- Build and deployment process

### 2. Charm Tree Analysis (Optional)

If `$1` is provided (e.g., "dawn_archery" or "water_socialize"), investigate that specific charm tree:

- Find the corresponding .mmd file
- Understand the charm hierarchy and prerequisites
- Identify any subgraphs and their purposes
- Count total charms in the tree
- Note the source books referenced
- Identify entry-level charms (no prerequisites)

### 3. Share Your Findings

Present what you learned about the documentation structure and/or specific charm tree in a clear, helpful way.

## Examples

- "/prime" - general overview of the documentation
- "/prime dawn_archery" - detailed analysis of Dawn Archery charm tree
- "/prime water_socialize" - detailed analysis of Water Aspect Socialize charms
